我解释一下：

  • connection name:连接名字随便起，这个无所谓
  • hostname:是你的ip地址，如果你是云服务器，你填上自己的公网地址就行（我是某云）
  • port:记得填上你的映射端口号 比如我们例子使用的是：13306
  • username :是root
  • password:你可以先输入上，如果不输入的时候，会提示你输入密码的
